# M1 retro — the loop closes, then meets a player

**Milestone:** M1 (C1–C10, per [`M1-loop.md`](../plans/M1-loop.md)) · **Signed off:** flywheel gate green + 30-turn soak clean (2026-07-10) · **First live play:** same day, "The Thread-Reader's Ledger" (7th Prince AU)

## What shipped

The whole loop, whole-shape (axiom 8): Sakkan blind scorer → SZ conductor + research
pipeline + compiler → Layout/KA/Compositor turn engine → Direction (Director, Pacer,
arcs, seeds, sessions) → drift band → channels + Series Bible → flywheel gate + soak.

## What the soak caught — engine-defect classes worth keeping named

1. **Adaptive thinking eats maxTokens** (five sightings: conductor, OSP, KA, sakuga,
   cost model). Headroom is now explicit everywhere; the class is priced in.
2. **API strict output guarantees the grammar, not the schema** — nested enums/ranges
   leak. `callStructured` gained its one corrective retry; code filters after.
3. **Row-terminal discipline** — a timeout that walks past a live turn corrupts every
   downstream read. A turn is only "past" when its row is terminal.
4. **Module-instance splits** — dynamic-importing the runtime in a suite that statically
   imports it forks the event bus. Static imports only.
5. **Grammar compiler caps at 24 optional params** — DirectorOutput slimmed to pairs.

## What the soak structurally could not catch — the live lesson

Every failure in the first live session was **environment or presentation, never the
engine**. The soak runs headless against local env; the player runs a browser against
Railway. That gap produced the whole day:

- **Env parity:** Railway production lacked `VOYAGE_API_KEY`. Research died at the last
  stage twice ($0.25), and the shared `aidm_v5` database masked *where* he was playing —
  two hours of local forensics against a server his traffic never touched.
- **The post-compile void:** the pilot rides turn 1, turn 1 waits for input, and the
  play view rendered silence after "we begin." Read as "SZ failed."
- **Stale Clerk token:** a 2-hour SZ tab 401'd the compile click; the UI treated 401 as
  final.
- **Silent deep think:** a sakuga turn thought for 3+ minutes behind a frozen
  "writing…" line — indistinguishable from a hang.
- **Presentation of honesty:** the §5.4 assertion notice, correct in substance, rendered
  twelve truncated writes as debug spew mid-story.
- **Dropped streams:** the browser SSE died during long server work (research, compile)
  while the server finished fine; the UI surfaced raw network errors.

**Carried lesson:** M2's definition of done needs an environment-parity check (deployed
env vars diffed against the schema) and a presentation pass on every player-facing
surface. A soak proves the engine; it cannot prove the experience.

## What worked live better than expected

- **The craft held the contract.** The pilot cold-open executed the SZ brief (hidden
  power as boredom, understatement discipline, anonymity preserved). The world-building
  turn metabolized a wall of player assertions into fiction instead of exposition.
- **§5.4 ingestion earned its keep** — player assertions minted a location, threads, and
  canon facts; the Chronicler independently admitted NPCs (Tomas, then Casimir Thoss)
  and enriched them turn-over-turn. The Red Sash design gap's fix direction is
  vindicated by watching the assertion path work where intent-gating allows it.
- **Resilience paths fired for real and held:** the commit_scene trailer dropped once on
  a 12.3k-token Opus generation and the probe fallback reconstructed the sidecar;
  research existence-refusal guarded honestly when config broke.
- **Sakkan read sanely** from turn 2 (register 6, epistemics 6–7, agency 7 against
  calibration; single-sample drift flags only).

## Walk-backs (recorded, per §15 spirit)

1. **"Poisoned env cache" theory retracted.** The dev server was never broken; the
   fingerprint I derived was Railway's genuinely-partial env. The env self-heal stays as
   cheap hardening, with its comment rewritten to the true rationale.
2. **"Calibration findings" mislabel.** The Red Sash miss was a *design gap* (intent
   gate narrower than §5.4 doctrine), not calibration; the douga threshold miss is
   calibration. The user's question exposed the sloppy taxonomy.
3. **Dockerfile rule_library worry retracted** — Next output tracing carries the YAML
   into standalone; research and settei proved it on Railway before I finished worrying.

   defer affordance; age opportunistic) — user directed this to the M2 plan.
4. **Compiler extraction misparse class** — "never as a fourth-wall voice" compiled to

5. **Douga threshold calibration** — needs a labeled set; golden douga seed waits on it.
6. **TTFT** — real targets vs aspirational; the heartbeat mitigates, doesn't solve.

   review the rate before hardening prompts.
8. **Director beat-shape variety** — both live turns ran task → quiet impossibility →
